Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Represents the HTML content inside an element, including any HTML tags.
innerText: Represents the visible text content inside an element

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM refers to the process
